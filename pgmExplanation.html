Who is Ishmael?

Call me Ishmael.—Moby Dick

In November, 2016, James Ford came home from class to meet a surprise that would change his life forever. On that day, he was gifted with a child, and I a brother. Ishmael never became physically impressive, but he quickly developed mastery of technology, integrating himself with the GroupMe API and adding functionality to all of the group chats our friend group created. Ever since he came into our lives, he has made them nothing but better.

The Bot

On the server side, Ishmael works through a Node.js server on Heroku, receiving and responding to different messages. For instance, Ishmael allows people to integrate Reddit into GroupMe, easily taking content and sending it to the group chat. In general, Ishmael provides a user-friendly interface for a plethora of responses to be defined by new users with no programming experience. Trigger words and corresponding messages are stored externally in a .JSON, though the data is loaded into a variable entitled ResponseArray, which contains objects like the following:


                    {
                        text:   'andy mineo'
                        response: {
                            type:       'direct',
                            message:    'I AIN'T DONE!',
                            attachments:[]
                        }
                    }

Typically, however, the bot is used to respond to certain commands, denoted by the addition of a '/' before a word. On input, the message is processed and utilizes whatever function is specified in the .JSON file. For example, if a user types the following:

/r dankmemes top

The response would then look something like this:


                    [Image]
                    [Title] Look at this image!

This is currently programmed to first check the title for profanity, and if any is found, move on to the next post. Ishmael also uses Optical Character Recognition (a method of recognizing text in images) to censor the images themselves for profanity. This ensures that the messages sent to the group chat remain clean. He does this until the 10th post he finds is profane, at which point he gives up, to avoid an endless loop on NSFW subreddits.
